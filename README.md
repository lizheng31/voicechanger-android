# VoiceChanger - 语音变声器

## 项目简介
VoiceChanger 是一个 Android 语音变声应用，允许用户录制和编辑音频，添加有趣的声音效果。

## 项目结构

```
app/src/main/
├── java/com/mengmeng/voicechager/
│   ├── api/                    # API 相关
│   ├── models/                 # 数据模型
│   ├── services/              # 服务组件
│   ├── utils/                 # 工具类
│   └── MainActivity.java      # 主界面
└── res/                       # 资源文件
```
