 # 语音变声器

一款 Android 平台的语音变声应用,支持录音和 AI 变声功能。

## 功能特点

- Material Design 3 界面设计
- 录音功能(支持暂停/继续)
- AI 变声处理
- 音频文件管理
- 适配 Android 10+ 存储权限

## 技术栈

- UI: Material Design 3
- 网络: Retrofit2 + OkHttp3
- 音频: MediaRecorder/MediaPlayer
- 权限: PermissionX
- 存储: 应用专属目录
